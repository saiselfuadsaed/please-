العناصر الهيكلية الأساسية في HTML

الصفحةالرئيسيه العناوين

<html>

العنصر الجذري الذي يحتوي على جميع عناصر HTML الأخرى. يُستخدم لتحديد اللغة الافتراضية للصفحة.

<!DOCTYPE html>
<html lang="ar">
  <head>
    <meta charset="UTF-8">
    <title>مثال</title>
  </head>
  <body>
    <h1>مرحبًا بالعالم!</h1>
  </body>
</html>

<head>

يحتوي على بيانات التعريف (metadata) مثل العنوان، الميتا داتا، الروابط الخارجية، وأي مراجع أخرى تحدد إعدادات أو موارد الصفحة.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>عنوان الصفحة</title>
  <link rel="stylesheet" href="styles.css">
  <script src="script.js" defer></script>
</head>

<body>

يحتوي على كل المحتوى المرئي على الصفحة مثل النصوص، الصور، الروابط، الأزرار، الجداول، إلخ.

<body>
  <h1>مرحبًا بكم في موقعي</h1>
  <p>هذا نص داخل جسم الصفحة.</p>
  <img src="image.jpg" alt="وصف الصورة">
</body>

العناصر الهيكلية التفصيلية

<!DOCTYPE html>

يحدد نوع وثيقة HTML. يضمن أن يعرض المتصفح الصفحة بشكل صحيح ومتناسق.

<!DOCTYPE html>
<html>
  <!-- هنا يبدأ المستند -->
</html>

<title>

يُستخدم لعرض عنوان القصير الذي يظهر في شريط العنوان في المتصفح.

<head>
  <title>عنوان الصفحة</title>
</head>

<meta>

يوفر معلومات وصفية عن الصفحة مثل الوصف، الكلمات المفتاحية، وترميز الأحرف.

<head>
  <meta charset="UTF-8">
  <meta name="description" content="مثال عن استخدام عناصر الميتا في HTML.">
  <meta name="keywords" content="HTML, CSS, برمجة">
</head>

<link rel="stylesheet" href="styles.css">

يربط الصفحة بملف CSS خارجي لتنسيق الصفحة.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

<script src="script.js" defer></script>

يربط الصفحة بملف JavaScript خارجي ليضيف التفاعلية إلى الصفحة.

<head>
  <script src="script.js" defer></script>
</head>

<h1> إلى <h6>

تُستخدم لتحديد العناوين من H1 إلى H6، مع H1 تكون العنوان الرئيسي والأكبر.

<body>
  <h1>عنوان رئيسي</h1>
  <h2>عنوان فرعي</h2>
  <h3>عنوان أصغر</h3>
</body>

<p>

عنصر الفقرة لتضمين نصوص في الصفحة.

<body>
  <p>هذه فقرة نصية.</p>
</body>

<img src="image.jpg" alt="وصف الصورة">

عنصر الصورة لتضمين الصور في الصفحة.

<body>
  <img src="image.jpg" alt="وصف الصورة">
</body>

<a href="https://example.com"> رابط (Link) عناصر إضافة</a>
